Stylus v0.1.0 User's Guide

Introduction to Stylus

The stylus tool is a multi-tool for a variety of tasks, and an incubator for new capabilities.

  • Scripted drawing to PNG images using the JavaFX GraphicsContext
  • Scripted drawing to monospaced text, for output to the terminal
  • Working with fictional calendar systems
  • Working with "historical" events and timelines

Some of these features, once mature, will become part of the Joe language's standard library.

Change Log

Changes in 0.1.0

  • Imported basic infrastructure and tool framework from pen, leaving all JTcl-based code behind.

joe.text package

The joe.text package contains APIs for advanced text formatting, and particularly for the formatting of monospace text for display in the terminal.

Types

  • Glyph
  • TextCanvas

Glyph type (joe.text)

TextCanvas type (joe.text)

TextCanvas is a canvas for drawing diagrams using monospaced text for output to the terminal. It can be thought of as a two-dimensional array of character cells, with (0,0) at the top-left with columns extending to the left and rows extending down. Characters and strings can be inserted at any (column, row) cell; the canvas will expand automatically.

TextCanvas Initializer

TextCanvas() → TextCanvas

Creates a new, empty TextCanvas of size [0, 0].

Methods

textCanvas.asText()

textCanvas.asText() → String

Returns the contents of the canvas as a String.

textCanvas.fill()

textCanvas.fill(char, column, row, width, height) → this

Fills the character cells in the given region with the given character.

textCanvas.get()

textCanvas.get(column, row) → String

Gets the character at (column, row) as a String

textCanvas.height()

textCanvas.height() → Number

Returns the height of the canvas in rows.

textCanvas.put()

textCanvas.put(column, row, text) → this

Writes the text to the canvas starting at (column, row) and extending to the right. No provision is made for multiline text.

textCanvas.putDown()

textCanvas.putDown(column, row, text) → this

Writes the text to the canvas starting at (column, row), and extending down.

textCanvas.putLeft()

textCanvas.putLeft(column, row, text) → this

Writes the text to the canvas starting at (column, row), and extending to the left, i.e., the text's rightmost character will be at (column, row). Text that extends to the left of column 0 is clipped.

textCanvas.putUp()

textCanvas.putUp(column, row, text) → this

Writes the text to the canvas starting at (column, row), and extending up, i.e., the text's rightmost character will be at (column, row). Text that extends above row 0 is clipped.

textCanvas.size()

textCanvas.size() → List

Returns the size of the canvas as a two-item list, [width, height].

textCanvas.toString()

textCanvas.toString() → String

Returns the canvas's string representation, which is not its content.

textCanvas.width()

textCanvas.width() → Number

Returns the width of the canvas in columns.
